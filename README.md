# My dot files

First install

```
$ curl -sS https://raw.githubusercontent.com/razbomi/.dotfiles/master/install.sh | zsh
```

Install updates after a change

```
$ cd ~/.dotfiles
$ ./install.sh
```

Developing loacally

```
$ INSTALL_DIR=../tmp ./install.sh
```
